Calculation Explanation

Runoff = Interflow + overland flow + baseflow
Interflow = Status * A, A varies from 0 to 1
Baseflow = constant
Total FC = ∑ Land use * Percentage
Temp SM(i+1)=Status Depth(i)+Rain(i)-Evaporation(i)-Interflow(i)-Baseflow
IF(Temp SM(i+1)>Total FC)
     Status Depth(i+1) = Total FC
ELSE IF(Temp SM(i+1)<0)
     Status Depth(i+1) = 0
ELSE
     Status Depth(i+1) = Temp SM(i+1)

What is the bucket model?

The bucket model is that we create a volume of soil in the form of a bucket. And for each time step, we will add rainfall and deduct evaporation to calculate the amount of water in the soil bucket. And at time zero, the assumption is that the bucket is full, so we have water depth=the field capacity.

How can I run the model?

After you download and open the model file (Online Excel doesn't enable Macros), you can change the distribution of land uses, and then run the model to get hydrological data and profile under different conditions. Considering thousands of data, the model simulates two kinds of units, including mm/day and mm/hour.

Why use VBA in Excel in Windows system?

For Windows system, there is no limit to use code to make applications do things. However, sometimes Mac cannot load visual basic for applications. For example, it's only possible to create a VBA Userform in previous versions of Mac Excel.

What is a Digital Elevation Model(DEM)?

A digital elevation model is a regularly-spaced bare-earth raster grid referenced to a common vertical datum. For hydrologic modelling, a DEM is used to delineate watersheds, calculate flow accumulation and find out flow direction.

How to use VBA in Excel?

In Visual Basic Editor(VBE), insert module and add VBA code. Save the file type as Excel Macro-Enabled Workbook. In order to run the added VBA code, press Alt+F8 to open the "Macro" dialog and select the wanted macro from the "Macro Name" list and click the "Run" button.
